## Retail Demand Forecasting System

Developed an end-to-end machine learning system for retail demand forecasting to predict future product sales and improve inventory management efficiency. The project aimed to minimize stock shortages and overstock situations by analyzing historical retail sales data and generating accurate demand predictions.

### Project Workflow

* Data Cleaning & Preprocessing
* Outlier Detection & Removal
* Feature Engineering
* Data Encoding & Scaling
* Model Training & Evaluation
* Forecast Performance Comparison

### Technologies & Tools

* Python
* Pandas
* NumPy
* Scikit-learn
* XGBoost
* LightGBM
* CatBoost
* Matplotlib

### Feature Engineering

Created and processed important features such as:

* Total Sales
* Day of Week
* Month
* Year

Applied:

* One-Hot Encoding
* Label Encoding
* StandardScaler normalization

### Machine Learning Models

Implemented and compared multiple regression and ensemble learning models:

* Linear Regression
* Decision Tree
* Random Forest
* Gradient Boosting
* AdaBoost
* XGBoost
* LightGBM
* CatBoost

### Results

| Model             | R² Score |
| ----------------- | -------- |
| CatBoost          | 0.9915   |
| LightGBM          | 0.9911   |
| Random Forest     | 0.9910   |
| XGBoost           | 0.9905   |
| Gradient Boosting | 0.9874   |
| Decision Tree     | 0.9823   |

CatBoost achieved the best overall forecasting performance with the highest prediction accuracy and lowest error metrics.
